# Banking Management System

A console-based Banking Management System built in Java, designed to simulate real-world banking operations with secure account management and transaction handling.

## Features

### Login & Account Management

* User Registration (Create Account)
* Secure Login System
* Separate User and Admin Panels
* Account Status Verification (Active/Disabled)
* Login Attempt Protection (Account blocked after multiple failed attempts)
* Help Center for Account Recovery and Unlock Requests
